# 🎵 Spotify Clone

A dynamic and responsive **Spotify Clone** web application built using **HTML, CSS, JavaScript**, and **Bootstrap**. This project mimics the core UI/UX features of Spotify, allowing users to browse music, control playback, and enjoy an immersive music player interface.

---

## 🚀 Features

- 🎧 Beautiful, responsive music player layout
- 🟢 Play/Pause functionality
- ⏭️ Next/Previous track controls
- 📂 Custom playlist with predefined songs
- 🔊 Interactive progress and volume controls
- 📱 Mobile-friendly design using Bootstrap

---

## 🛠️ Tech Stack

- **HTML5** – Structuring the web pages  
- **CSS3** – Styling and animations  
- **JavaScript** – Handling logic and player functionalities  
- **Bootstrap** – Ensuring responsive design and layout

---

## 🔧 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/spotify-clone.git
   ```
2. Navigate to the project folder:
   ```bash
   cd spotify-clone
   ```
3. Open `index.html` in your browser to view the app.

> ⚠️ No backend/server setup is required. This is a frontend-only project.
